# 🍎 Eating ESP Mod

**Minecraft Fabric Mod — 1.21.1**

Показывает ESP прямо на туловище игрока, который ест или пьёт — видно даже сквозь стены!

---

## Что показывает мод

| Предмет | Цвет ESP |
|---|---|
| Золотое яблоко / Зачарованное | 🟡 Золотой |
| Зелья (все виды) | 🟣 Фиолетовый |
| Золотая морковка | 🟡 Жёлто-золотой |
| Мёд / Медовая бутылка | 🟠 Оранжевый |
| Ведро молока | ⬜ Белый |
| Плод хоруса | 🟣 Розово-фиолетовый |
| Гнилая плоть, паучий глаз, отравленная картошка | 🔴 Красный (опасно!) |
| Подозрительное рагу | 🟢 Зелёный |
| Обычная еда (хлеб, мясо, рыба, ягоды...) | 🟢 Зелёный |

---

## Что отображается в ESP

- **Иконка предмета** — прямо над центром туловища
- **Пульсирующая анимация** — иконка "дышит"
- **Прогресс-бар** — показывает как далеко зашла анимация поедания
- **Рендер сквозь стены** — работает через любые блоки

## Установка

### Требования
- Minecraft **1.21.1**
- [Fabric Loader](https://fabricmc.net/) `≥ 0.15.0`
- [Fabric API](https://modrinth.com/mod/fabric-api)

### Шаги
1. Установите Fabric Loader для Minecraft 1.21.1
2. Скачайте Fabric API и положите в папку `mods/`
3. Положите `eating-esp-1.0.0.jar` в папку `mods/`
4. Запустите игру!

---

## Сборка из исходников

```bash
# Клонируй репо
git clone ...
cd eating-esp-mod

# Собери
./gradlew build

# JAR будет в: build/libs/eating-esp-1.0.0.jar
```

---

## Как работает

Мод использует `WorldRenderEvents.AFTER_ENTITIES` от Fabric API.  
После рендера всех сущностей, мод итерирует всех игроков в мире и для каждого кто `isUsingItem()` и держит consumable — рисует ESP через `RenderLayer` с выключенным depth test, что даёт эффект "сквозь стены".
